## Установка

```shell
curl https://sh.rustup.rs -sSf | sh
```

## Новый проект

```shell
cargo new имя_проекта
cd имя_проекта
```

## Сборка и запуск

```shell
cargo build
cargo run
```

## Тестирование

```shell
cargo test
```

## Полезные команды

```shell
cargo check       # Быстрая проверка
cargo fmt         # Форматирование кода
cargo clippy      # Подсказки и предупреждения
cargo doc --open  # Документация
```
